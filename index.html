Canada COVID-19 Vaccination Schedule

These visualizations illustrate the huge task to meet the Canadian government's goal of providing vaccinations for a target population, defined as age 18+ Canadians, by Sep 30, 2021.

  • Set of charts, one for Canada and each province or territory that has the following data:
    • Actual count of vaccine doses administered to-date across Canada as reported by provinces (blue bars).
    • Calculated count of dose administration remaining that is required to fully vaccinate age 18+ population by Sep 30, 2021 (gray bars).
  • One chart for Canada that has the following data:
    • Canada Pfizer & Moderna Vaccine Dose Delivery Forecast By Sep 30, 2021. This chart includes estimated daily vaccine dose deliveries (orange and red bars) and a cumulative vaccine dose total (black line).

COVID-91 actual vaccine data comes from the COVID-19 Canada Open Data Working Group who are doing an amazing job collating provincial COVID-19 data! The vaccine roll-out forecast is my own creation.

Additional Notes:

Edit Feb 5, 2021 - modified definition of "target population" to be vaccinated by Sep 30 from age 15+ to age 18+ population.

The total calculated count of required dose administration for Canada and each province is equal to their respective population x 2, minus the count of doses administered to-date. A daily dose count is then calculated by dividing the total by the number of days between the last actual reporting date and Sep 30, 2021.

Population x 2 is used because dose administration reporting records single doses but two doses are required for full vaccination using the available Pfizer and Moderna vaccines. This would be updated if and when single dose vaccines become available.

% of Target Population Doses Administered is defined as: ((Doses Administered To-Date / (Age 18+ Popluation x 2)) x 100. This is probably the best metric to gauge overall performance of vaccination program as each person requires 2 doses.
